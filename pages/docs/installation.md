# Installation

## Prerequisites

Node.js is required.

## Cloning from Github

``` bash
git clone https://github.com/nosenko-mi/sn-api.git
cd sn-api
npm start 
```

Expected output:

``` bash
Your server is listening on port 8080 (http://localhost:8080)
Swagger-ui is available on http://localhost:8080/docs
```
